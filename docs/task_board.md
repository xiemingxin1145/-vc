# 三国人生模拟器任务看板

## P0：必须先做

### 1. 事件插图显示

目标：年度事件弹窗显示对应插图。

验收标准：

- 高级事件有图。
- 普通事件无图时不报错。
- 弹窗布局不溢出。

### 2. 成就入口

目标：主界面或记录页面可以查看成就达成情况。

验收标准：

- 显示已达成和未达成。
- 使用 `SanguoAchievementEngine.unlockedAchievements(viewModel)`。
- 不改数据库。

### 3. 图库入口

目标：展示 `SanguoGallery.eventArts` 的六张插图。

验收标准：

- 能看到标题、图片、说明。
- 可以从标题页或游戏页进入。

## P1：增强玩法

### 4. 高级事件接入年度事件池

目标：每年事件有概率刷出 `SanguoContentPack.premiumEvents`。

验收标准：

- 20% 到 35% 概率刷出高级事件。
- 原普通事件仍然保留。
- 新事件结果能产生属性、金钱、名望、关系等变化。

### 5. 事件池扩展到 30 条

目标：降低重复感。

分类：

- 战争事件
- 官场事件
- 经商事件
- 名人互动事件
- 家族事件
- 城池事件

## P2：成品感

### 6. 结局列传强化

目标：游戏结束时像历史列传。

验收标准：

- 根据属性和经历生成称号。
- 显示一生总结。
- 结局等级更清晰。

### 7. 构建 APK

目标：跑通 debug APK。

命令：

```bash
./gradlew assembleDebug
```

验收标准：

- 能编译。
- 能安装。
- 能开局。
- 能推进年份。
- 能存档读取。
