# 三国人生模拟器 · 清理与重做清单

> 目标：删掉旧壳子和没接入的半成品，把游戏整理成“有图片、有角色创建、有官位成长、有可玩的战争系统”的版本。

## 0. 当前问题判断

- 当前 APK 更像“基础壳子”，不是完整新版。
- 图片素材索引已经存在，但事件弹窗和标题页没有真正显示。
- 战争扩展包已经存在，但目前偏“文字事件结算”，不是玩家期待的即时战争/战场操作。
- 人物创建太单薄：姓名、外貌、出身、祖籍、天赋都不够。
- 官位体系太少：文官、武官路线只有几档，缺少爬官爽感。
- 年度事件数量偏少，容易重复。

## 3. 即时战争 / 战场系统清单

### 3.1 当前战争问题

- [ ] 当前 `SanguoBattleExpansion.kt` 是剧情战斗事件，不是即时战争。
- [ ] 玩家看不到敌我兵力变化过程。
- [ ] 玩家不能布阵、冲锋、防守、撤退。
- [ ] “金戈伐图”页需要变成真正可玩的战场入口。

### 3.2 第一版战场目标：简易可玩即时战斗

- [ ] 新增战场状态：敌军兵力、我军兵力、士气、回合、战术日志。
- [ ] 增加战术按钮：
  - [ ] 全军冲锋
  - [ ] 稳守阵线
  - [ ] 弓兵齐射
  - [ ] 骑兵突击
  - [ ] 谋略火攻
  - [ ] 撤退保命
- [ ] 每次点击战术，立即结算一轮伤害和士气变化。
- [ ] 战斗过程显示滚动战报。
- [ ] 胜利后占领城池、涨声望、涨统一进度。
- [ ] 失败后损兵、降士气、扣健康。

### 3.3 兵种克制

- [ ] 步兵克弓兵。
- [ ] 弓兵克骑兵。
- [ ] 骑兵克步兵。
- [ ] 士气低会降低伤害。
- [ ] 训练高会提高稳定性。
- [ ] 武力影响单挑和冲锋。
- [ ] 统率影响全军战斗。
- [ ] 智谋影响火攻、奇袭、埋伏。

### 3.4 地图攻城流程

- [ ] 在 `MapTerritoryTab` 里选择城池。
- [ ] 增加按钮：侦察、征兵、攻城、外交劝降。
- [ ] 点“攻城”进入战场界面。
- [ ] 胜利后调用城池归属变更。
- [ ] 城池收入影响每年俸禄/税收。

## 7. 推荐执行顺序

1. 清理旧 APK / 旧文档 / 半成品标记。
2. 接入图片系统：标题页 + 事件弹窗。
3. 接入 `SanguoContentPack.premiumEvents`。
4. 重做战争入口：从“文字战斗”变成“战场界面”。
5. 扩展人物创建：姓名、字、号、外貌、祖籍、出身。
6. 扩展官位树。
7. 扩展年度事件库。
8. 重新构建 APK，并确认手机安装的是新版。
